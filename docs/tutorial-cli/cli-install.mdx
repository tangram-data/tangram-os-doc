# Install Tangram OS CLI



### Prerequisites

- **Java 11+**

### Download Tangram CLI Jar 

- Download [Tangram CLI JAR](https://github.com/tangram-data/tangram-os-doc/releases/download/tangram-cli-1.0.0/tangram-cli_2.13-1.0.0-all.jar) file
- Move it to your preferred directory


### Start Tangram OS CLI Shell with Java
```bash
java -jar ./tangram-cli_2.13-1.0.0-all.jar
```
