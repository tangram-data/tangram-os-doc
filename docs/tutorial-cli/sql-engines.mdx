# SQL Engines

The sql-engine command group allows you to manage and interact with SQL database engines in Tangram OS. 

### List available SQL engines:
```bash
tangram> local> sql-engine ls

id                                       | alias  | resource_name
-----------------------------------------+--------+--------------------------------------------------
1f50fb38a8020364255a12b4e113c92a32503c5e | duckdb | rn://org.duckdb/duckdb/Instance/tangram_os/duckdb

```

### Connect to a SQL engine to start interactive SQL shell:
```python
# Example: sql-engine connect duckdb
sql-engine connect <sql-engine-id or alias>
```

### Get SQL engine alias:
```bash
sql-engine alias get <sql-engine-id>
```
