# SQL Engines

The sql-engine command group allows you to manage and interact with SQL database engines in Tangram OS. 

### List available SQL engines:
```bash
tangram sql-engine ls -w tangram_demo

id                                       | name                                             
-----------------------------------------+--------------------------------------------------
62b6f2822389fe90fa5a4cb02882bcc3f7241997 | rn://org.duckdb/duckdb/Instance/tangram_os/duckdb

```

### Connect to a SQL engine and open interactive shell:
```bash
sql-engine connect 62b6f2822389fe90fa5a4cb02882bcc3f7241997
```

### Get SQL engine alias:
```bash
sql-engine alias get 62b6f2822389fe90fa5a4cb02882bcc3f7241997
```
